# Stone Paper Scissor

A simple React + Vite rock-paper-scissors game with a basic UI.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Netlify deploy

This project is ready for Netlify with the included `netlify.toml`.

- Build command: `npm run build`
- Publish directory: `dist`

If you connect the GitHub repo in Netlify, it should detect these settings automatically.
